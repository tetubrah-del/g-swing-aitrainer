# 🧠 フェーズ別診断 専用RAG（決定版）

※ フェーズ共通思想 → 各フェーズ定義の順

---

## 0️⃣ フェーズ別診断の共通思想（必須）

- 各フェーズは0〜20点で評価する
- 評価は「形」ではなく「次フェーズへの影響度」で行う
- 小さな個性は減点しない
- 再現性を壊す要素のみを強く減点する

### 点数帯の共通定義（全フェーズ共通）

| 点数帯   | 定義                |
| ----- | ----------------- |
| 16–20 | 次フェーズに悪影響なし／再現性高い |
| 11–15 | 軽微なズレあり／結果に影響しにくい |
| 6–10  | 明確なズレあり／次フェーズに悪影響 |
| 0–5   | 破綻レベル／再現性が著しく低い   |

# ① Address（構え）

## 評価観点（5）

1. バランス（前後・左右）
2. 前傾姿勢の安定
3. 手元位置（体との距離）
4. スタンス幅・向き
5. リラックス度（力み）

## OK / NG 具体例

**OK**

- 体重が足裏中央付近
- 前傾が自然で背中が安定
- 手元が太もも付近で一定
- スタンスがターゲットに対し自然
- 肩・腕に過度な力みなし

**NG**

- つま先／かかと体重
- 前傾が浅すぎる／深すぎる
- 手元が体から極端に離れる
- クローズ／オープンが極端
- 肩がすくんで力みが強い

## 点数帯の目安

- 16–20：自然で再現性高い
- 11–15：個性レベルのズレ
- 6–10：次フェーズに影響
- 0–5：構え直しが必要

## ハードペナルティ

- 明らかな前後バランス崩壊
- 手元が極端に高い／低い

## 中上級（70+）最低条件

- 前後バランスが安定
- 手元位置が毎回一定
- 満たさない場合：上限14点

---

# ② Backswing（始動〜途中）

## 評価観点（5）

1. 始動のスムーズさ
2. 体と腕の同調
3. 手元の軌道安定
4. 上半身の回転量
5. バランス保持

## OK / NG

**OK**

- ワンピース的な始動
- 腕が体の正面を保つ
- 手元が一定軌道
- 胸が自然に回る
- 下半身が安定

**NG**

- 手だけで急始動
- 腕が外に流れる
- 手元が上下に暴れる
- 上体が突っ込む
- 体重が左右に流れる

## 点数帯

- 16–20：同調が取れている
- 11–15：軽微なズレ
- 6–10：次フェーズに影響
- 0–5：再現性なし

## ハードペナルティ

- 腕が体の外に大きく外れる
- 始動でバランス崩壊

## 中上級最低条件

- 始動が急激でない
- 腕が体の正面にある
- 未達：上限13点

---

# ③ Top（トップ）

## 評価観点（5）

1. 回転量の適正
2. 腕と体の位置関係
3. 手元高さ
4. バランス
5. 張りすぎ・詰まりすぎの有無

## OK / NG

**OK**

- 回り過ぎず止まれる
- 腕が体の正面
- 手元が肩〜耳付近
- 軸が安定
- 張り感が自然

**NG**

- 回り過ぎ／浅すぎ
- 腕が体の後ろ
- 手元が極端に高低
- 上体が流れる
- 力みで止まれない

## 点数帯

- 16–20：切り返し準備完了
- 11–15：個性範囲
- 6–10：切り返しに悪影響
- 0–5：破綻

## ハードペナルティ

- トップが極端に深い／浅い
- バランスを崩して立て直す

## 中上級最低条件

- 止まれるトップ
- バランス保持
- 未達：上限12点

---

# ④ Downswing（切り返し〜下ろし）

## 評価観点（6）

1. 下半身先行
2. 上半身の待ち
3. 腕の位置
4. クラブの下り方
5. 体の開き具合
6. バランス

## OK / NG

**OK**

- 下半身から動く
- 胸が一瞬待つ
- 腕が体の正面
- 自然に下りる
- 開きが抑えられる

**NG**

- 上半身先行
- 全身同時スタート
- 腕が外に放り出される
- 無理に下ろす
- 早開き
- アウトサイドイン（カット軌道）／外から下りる

## 点数帯

- 16–20：最重要成功フェーズ
- 11–15：許容範囲
- 6–10：明確な動的異常
- 0–5：ほぼ失敗確定

## ハードペナルティ

- 上半身先行が明確
- 外から下りる

## 中上級最低条件

- 下半身先行
- 胸の待ち
- 未達：上限10点

---

# ⑤ Impact（インパクト）

## 評価観点（4）

1. 体勢の安定
2. 入射角
3. 手元位置
4. 体と腕の関係

## OK / NG

**OK**

- 体が崩れない
- 適正入射
- 手元が体の近く
- 体と腕が同調

**NG**

- 体勢が崩れる
- すくい打ち
- 手元が浮く
- 手だけ当てる

## 点数帯

- 16–20：再現性高い
- 11–15：軽微なズレ
- 6–10：結果が不安定
- 0–5：ミス直結

## ハードペナルティ

- 体勢崩壊
- 明確なすくい

## 中上級最低条件

- 体勢が安定
- 未達：上限12点

---

# ⑥ Finish（フィニッシュ）

## 評価観点（4）

1. バランス
2. 体の向き
3. 振り切り
4. 静止性

## OK / NG

**OK**

- 片足で安定
- 体が目標向き
- 振り切れている
- 静止できる

**NG**

- ふらつく
- 体が止まらない
- 振り切れない
- 体勢を崩す

## 点数帯

- 16–20：完成度高い
- 11–15：問題なし
- 6–10：体力・再現性不足
- 0–5：崩壊

## ハードペナルティ

- 立っていられない
- バランス喪失

## 中上級最低条件

- 静止可能
- 未達：上限14点

---

## 🔚 このRAGを入れると起きる変化

- フェーズ別点数が意味を持つ
- 減点が「感覚」ではなく「理由付き」になる
- 後段補正（B）が作りやすくなる
